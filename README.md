## 🚀 Deploy on Koyeb (Free)

### Step 1 — Push to GitHub
1. Create a GitHub repo called `ScottyMd`
2. Upload all files to it
3. session/ and .env are gitignored ✅

### Step 2 — Deploy on Koyeb
1. Go to [koyeb.com](https://koyeb.com) → sign up free
2. New App → GitHub → select ScottyMd repo
3. Set environment variables:
   - `OWNER_NUMBER` = `263788114185`
   - `NODE_VERSION` = `18`
   - `PORT` = `8000`
4. Set health check path: `/ping`
5. Deploy

### Step 3 — Pair
1. Open your Koyeb URL in browser
2. Enter your WhatsApp number
3. Get pairing code
4. WhatsApp → Settings → Linked Devices → Link a Device → Enter code ✅

### Step 4 — Save SESSION_ID
1. Once online, send `.session` to bot
2. Copy the file contents
3. Set as `SESSION_ID` env variable on Koyeb
4. Redeploy — never re-pair again ✅
